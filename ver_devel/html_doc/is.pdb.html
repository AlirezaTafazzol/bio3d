Is an Object of Class ‘pdb’?

Usage

is.pdb(x)

Arguments

x
an R object to be tested

Description

Checks whether its argument is an object of class ‘pdb’.

Details

Tests if x is an object of class ‘pdb’, i.e. if x has a “class” attribute equal to pdb.

Value

TRUE if x is an object of class ‘pdb’ and FALSE otherwise

Examples

# Read a PDB file
pdb <- read.pdb("1BG2")



  Note: Accessing online PDB file
  HEADER    MOTOR PROTEIN                           04-JUN-98   1BG2               



is.pdb(pdb)



[1] TRUE
