Run Length Encoding with Indices

Description

Compute the lengths, values and indices of runs of equal values in a vector. This is a modifed version of base function rle().

Details

Missing values are regarded as unequal to the previous value, even if that is also missing.

inverse.rle() is the inverse function of rle2() and rle(), reconstructing x from the runs.

Value

rle() returns an object of class "rle" which is a list with components: lengthsan integer vector containing the length of each run. valuesa vector of the same length as lengths with the corresponding values.

Examples

x <- rev(rep(6:10, 1:5))
rle(x)

rle2



function (x) {
  ## This is a modified version of base function "rle()"
  if (!is.vector(x) && !is.list(x)) 
    stop("'x' must be an atomic vector")
  n <- length(x)
  if (n == 0L) 
    return(structure(list(lengths = integer(), values = x, inds=integer), class = "rle2"))
  y <- x[-1L] != x[-n]
  i <- c(which(y | is.na(y)), n)
  structure(list(lengths = diff(c(0L, i)), values = x[i], inds=i), class = "rle2")
}




## lengths: int [1:5] 5 4 3 2 1
## values : int [1:5] 10 9 8 7 6
## indices: int [1:5] 5 9 12 14 15
