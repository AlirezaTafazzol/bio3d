List all Functions in the bio3d Package

Usage

lbio3d()

Description

A simple shortcut for ls("package:bio3d").

Value

A character vector of function names from the bio3d package.

References

Grant, B.J. et al. (2006) Bioinformatics 22, 2695--2696.
