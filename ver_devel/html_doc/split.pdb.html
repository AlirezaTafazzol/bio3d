Split a PDB File Into Separate Files, One For Each Chain.

Usage

pdbsplit(pdb.files, path = "split_chain/", multi = FALSE)

Arguments

pdb.files
a character vector of PDB file names.
path
output path for chain-split files.
multi
logical, if TRUE atom records for all models in multi-model files are splitted.

Description

Split a Protein Data Bank (PDB) coordinate file into new separate files with one file for each chain.

Details

This function will produce single chain PDB files from multi-chain input files.

Multi model atom records will be splitted into individual PDB files if multi=TRUE, else they are omitted.

Value

Returns a character vector of file names.

References

Grant, B.J. et al. (2006) Bioinformatics 22, 2695--2696.

For a description of PDB format (version3.3) see: http://www.wwpdb.org/documentation/format33/v3.3.html.

Note

To Be Improved.

Examples

  ## Fetch PDBs
  pdb.ids <- c("1YX5", "3NOB", "1P3Q", "2D3G","1F9J")
  outdir <- paste(tempdir(), "/raw_pdbs", sep="")
  raw.files <- get.pdb(pdb.ids, path = outdir)

  ## Split PDBs by chain ID and multi-model records
  all.files <- pdbsplit(raw.files,
            path =paste(outdir, "/split_chain", sep=""), multi=TRUE )
